FC_CancelFunction

Name

FC_CancelFunction - cancel a function running in parallel

Syntax 

CK_RV FC_CancelFunction(
  CK_SESSION_HANDLE hSession
);

Parameters

hSession
[in] session handle.

Description

Parallel functions are not implemented. FC_CancelFunction is a legacy function that simply returns CKR_FUNCTION_NOT_PARALLEL.

Return value

FC_CancelFunction always returns CKR_FUNCTION_NOT_PARALLEL.
