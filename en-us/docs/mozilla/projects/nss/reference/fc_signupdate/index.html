FC_SignUpdate

Name

FC_SignUpdate - process the next block of a multi-part signing operation.

Syntax 

CK_RV FC_SignUpdate(
  CK_SESSION_HANDLE hSession,
  CK_BYTE_PTR pPart,
  CK_ULONG usPartLen
);

Parameters

hSession
[in] session handle.
pPart
[in] pointer to the next block of the data to be signed.
usPartLen
[in] length of data block in bytes.

Description

FC_SignUpdate starts or continues a multi-part signature operation. One or more blocks may be part of the signature. The signature for the entire message is returned by a call to FC_SignFinal.

A user must log into the token (to assume the NSS User role) before calling FC_SignUpdate.
