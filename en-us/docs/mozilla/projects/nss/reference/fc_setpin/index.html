FC_SetPIN

Name

FC_SetPIN - Modify the user's PIN.

Syntax 

CK_RV FC_SetPIN(
  CK_SESSION_HANDLE hSession,
  CK_CHAR_PTR pOldPin,
  CK_ULONG ulOldLen,
  CK_CHAR_PTR pNewPin,
  CK_ULONG ulNewLen
);

Parameters

FC_SetPIN takes five parameters:

hSession
[Input] the session's handle
pOldPin
[Input] points to the old PIN.
ulOldLen
[Input] the length in bytes of the old PIN.
pNewPin
[Input] points to the new PIN.
ulNewLen
[Input] the length in bytes of the new PIN.

Description

FC_SetPIN modifies the PIN of the user. The user must log into the token (to assume the NSS User role) before calling FC_SetPIN.

Return value

CKR_OK
